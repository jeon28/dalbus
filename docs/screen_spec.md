# 전체 IA (Information Architecture)

```
달버스 (dalbus.kr)
│
├─ 🏠 공개 영역 (Guest + User + Admin)
│   ├─ [SCR_LAND_001] 랜딩 페이지 (메인)
│   ├─ [SCR_SHOP_001] 상품 목록
│   ├─ [SCR_SHOP_002] 상품 상세 / 기간 선택
│   ├─ [SCR_NOTICE_001] 공지사항 목록
│   ├─ [SCR_NOTICE_002] 공지사항 상세
│   ├─ [SCR_FAQ_001] FAQ
│   └─ Footer: 이용약관 / 개인정보처리방침 / 사업자 정보
│
├─ 🔐 인증 영역 (Guest → User)
│   ├─ [SCR_AUTH_001] 로그인
│   ├─ [SCR_AUTH_002] 회원가입
│   ├─ [SCR_AUTH_003] 비밀번호 재설정 요청
│   └─ [SCR_AUTH_004] 비밀번호 재설정 완료
│
├─ 💳 결제 영역 (User)
│   ├─ [SCR_PAY_001] 결제 진행 (PG 팝업)
│   ├─ [SCR_PAY_002] 결제 완료
│   └─ [SCR_PAY_003] 결제 실패
│
├─ 👤 마이페이지 (User)
│   ├─ [SCR_MY_001] 구독 현황 (계정 정보 확인)
│   ├─ [SCR_MY_002] 주문 내역
│   └─ [SCR_MY_003] 프로필 수정
│
├─ 🛡️ 관리자 (Admin)
│   ├─ [ SCR_ADM_001] 주문 관리 (Multi-Select 필터, 전화번호 검색, Excel Export)
│   ├─ [SCR_ADM_002] Tidal 계정 관리 (마스터/슬롯, Excel Export)
│   ├─ [SCR_ADM_003] 서비스 관리
│   ├─ [SCR_ADM_004] 회원 관리
│   ├─ [SCR_ADM_005] 공지/FAQ 관리
│   ├─ [SCR_ADM_006] 운영 설정 (계좌/알림)
│   └─ [SCR_ADM_007] Q&A 관리
│
└─ ⚠️ 에러
    └─ [SCR_ERR_403] 접근 제한
```

* 이메일/비밀번호 기반 로그인

### 화면 전환
* 로그인 성공 → 이전 페이지 (returnUrl) 또는 SCR_LAND_001
* 'Google로 계속하기' → Supabase Auth Google OAuth 플로우
* '비밀번호를 잊으셨나요?' → SCR_AUTH_003
* '회원가입' → SCR_AUTH_002

### 유효성 검사
* 이메일 형식 불일치: 인라인 에러 "올바른 이메일 형식을 입력해주세요"
* 로그인 실패: 공통 에러 "이메일 또는 비밀번호가 올바르지 않습니다" (보안)

* '로그인' 링크 → SCR_AUTH_001

### 유효성 검사 (인라인 실시간)
* 이메일 형식 / 중복 체크

* 휴대폰 번호 형식 (010-xxxx-xxxx)
* 약관 미동의 시 가입 버튼 비활성화
