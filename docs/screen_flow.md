# Screen Flow (사용자 및 관리자 흐름도)

## 1. 회원 / 비회원 주문 프로세스

### 1-1. 비회원 (Guest) 주문 흐름
> **정책**: 비회원은 결제 전 가입 또는 로그인을 유도 받습니다. (현재 MVP 기준)

1. **[랜딩 페이지]**
   - 서비스 둘러보기, 공지사항/리뷰 확인
   - `[시작하기]` 클릭 → **[서비스 목록]** 이동

2. **[서비스 목록]**
   - 상품 카드(예: Tidal, YT Premium) 선택 → **[서비스 상세]** 이동

3. **[서비스 상세]**
   - 기간(1개월/3개월) 선택
   - `[결제하기]` 클릭
   - **(분기)** 비회원 감지 시 → **[로그인/회원가입]** 페이지로 리다이렉트 (returnURL 포함)

4. **[회원가입/로그인]**
   - 로그인 또는 회원가입 완료
   - 완료 후 다시 **[서비스 상세]** 또는 **[결제 진행]** 화면으로 복귀

5. **[결제 진행]**
   - 최종 금액 확인 및 PG사 결제창 호출
   - 결제 완료

6. **[결제 완료]**
   - 주문 성공 메시지, 배정 대기 안내
   - `[마이페이지로 이동]` 버튼 클릭

---

### 1-2. 회원 (Member) 주문 흐름

1. **[로그인 상태]**
   - **[서비스 목록]** → **[서비스 상세]** → 기간 선택
   - `[결제하기]` 클릭 → **[결제 진행]** (PG사 창)
   - 결제 완료 → **[결제 완료]** 페이지

---

## 2. Q&A (문의하기) 프로세스

### 2-1. 문의 글 작성

- **[Q&A 목록]** 페이지 진입
  - `[문의하기]` 버튼 클릭 → **[문의 작성]** 페이지

- **(User Type: 회원)**
  - 작성자 이름: 자동 입력 (수정 불가)
  - 비밀번호: 입력 불필요
  - 제목/내용 입력
  - '비밀글' 체크 가능

- **(User Type: 비회원)**
  - 작성자 이름: **필수 입력**
  - 비밀번호: **필수 입력** (추후 수정/삭제용)
  - 안내 문구: "비회원은 비밀번호를 꼭 기억해주세요."
  - 제목/내용 입력
  - '비밀글' 체크 가능 (작성자와 관리자만 볼 수 있음)

- `[등록하기]` 클릭 → DB 저장 → **[Q&A 목록]**으로 이동

### 2-2. 문의 글 조회 및 수정

- **[Q&A 목록]**
  - '비밀글 제외' 필터 제공
  - 비밀글 클릭 시:
    - **본인(작성자)** 또는 **관리자**: 내용 바로 확인 (회원은 로그인 ID 비교)
    - **비회원 작성자**: **[비밀번호 확인]** 모달/입력창 노출 → 일치 시 내용 확인
    - **제3자**: "비밀글입니다" 안내 및 접근 불가

- **수정/삭제**
  - 회원: 본인 글인 경우 수정/삭제 버튼 활성화
  - 비회원: 상세 보기 상태(비밀번호 인증 후)에서 수정/삭제 가능

## 3. 관리자 (Admin) 프로세스

> **전제**: 관리자 계정으로 로그인 상태

### 3-0. 모바일 헤더 메뉴 개선
- 모바일 환경에서 브랜드 로고 상시 노출.
- '서비스', '공지사항', 'FAQ', 'Q&A' 메뉴를 가로 스크롤 형태로 메인 헤더에 상시 노출하여 접근성 강화.

### 3-1. 회원 관리
- **[관리자 대시보드]** → 상단 메뉴 **[회원정보]** 클릭
- **[회원 목록 화면]**
  - 가입된 모든 사용자 리스트 조회 (이름, 이메일, 가입일 등)

### 3-2. 서비스(상품) 관리
- 상단 메뉴 **[서비스 관리]** 클릭
- **[서비스 목록 화면]**
  - 등록된 상품 리스트 확인
  - `[서비스 추가]` 버튼 → **[신규 서비스 등록]** 폼
- **[서비스 수정 화면]** (목록에서 클릭)
  - 상품 정보 수정 (이름, 설명, 가격 등)
  - **[요금제(Plan) 관리]** 섹션: 기간별(1개월, 3개월 등) 옵션 추가/삭제/수정

### 3-3. 주문 관리
- 상단 메뉴 **[주문관리]** 클릭
- **[주문 목록 화면]**
  - 모든 내역에 **'이용기간(개월)'** 정보가 상시 노출되어 배정 시 확인 가능.

### 3-4. 공지사항 및 FAQ 관리
- 상단 메뉴 **[공지사항 관리]** 또는 **[FAQ 관리]** 클릭
- **[목록 및 작성]**: CRUD 기능 제공
- **[카테고리 관리]**: 동적으로 카테고리를 추가/삭제/수정 가능.

### 3-5. 운영 설정 (무통장 계정 관리)
- **[관리자 대시보드/설정]** 섹션
- 무통장 입금용 은행명, 계좌번호, 예금주를 직접 추가/삭제.
- 변경된 정보는 사용자 결제 페이지 및 성공 페이지에 즉시 반영.

### 3-6. Q&A 관리
- 상단 메뉴 **[Q&A 관리]** 클릭
- **[전체 문의 목록]**
  - 모든 문의 글(비밀글 포함) 조회 가능
  - 상태(미답변/답변완료) 확인
- **[답변 작성]**
  - 문의 내용 하단 `[답변하기]` 클릭 (또는 상세 진입)
  - 답변 내용 입력 후 등록 → 사용자 화면에 '답변완료' 및 답변 내용 게시
