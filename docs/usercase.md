# Share King (쉐어킹) - User Cases

## User Personas
- **The Cost-Saver**: Wants to use Tidal or Netflix but finds the individual subscription too expensive.
- **The Convenience-Seeker**: Wants a reliable service that manages the sharing group and account updates.

## Core User Journeys

### 1. New Subscription
1. User lands on Share King.
2. User selects "Tidal" service.
3. User sees price (e.g., 4,900 KRW/month) and clicks "Subscribe".
4. User authenticates via mobile (PortOne).
5. User completes payment.
6. User is redirected to a success page.
7. User receives a KakaoTalk notification when the account is ready.

### 2. Accessing Account Info
1. User goes to "My Page".
2. User clicks on "Active Subscriptions" -> "Tidal".
3. User sees the assigned ID and Password.
4. User logs into Tidal and enjoys the service.

### 3. Administrator Management
1. Admin logs into the dashboard.
2. Admin sees a new "Paid" order for Tidal.
3. Admin assigns an available Tidal family account profile to the user.
4. Admin clicks "Complete", which triggers an automated notification to the user.
