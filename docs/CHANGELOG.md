# Changelog - 2026.02.06 ~ 02.07

Dalbus 프로젝트의 주요 업데이트 및 변경 사항을 기록합니다.

---

## [Supabase 백엔드 통합]
- **인증 시스템**: Supabase Auth를 연동하여 실제 회원가입 및 로그인 기능 구현.
- **실시간 데이터**: `ServiceContext`를 통해 Supabase DB의 서비스 및 가격 정보를 실시간으로 연동.
- **주문 및 매칭**: 서비스 상세 페이지에서 실제 주문 생성 로직 구현 및 마이페이지에서 배정된 계정 정보 확인 가능.
- **어드민 대시보드**: DB 기반 실시간 매출 통계 및 주문 내역 조회 기능 구현.

## [UI/UX 개편]
- **화이트 테마 전환**: 기존 다크 모드에서 프리미엄 화이트 테마 및 슬레이트 블랙 포인트 디자인으로 전면 개편.
- **Glassmorphism 업데이트**: 밝은 배경에 최적화된 반투명 카드 및 부드러운 그림자 효과 적용.
- **모바일 퍼스트**: 전반적인 반응형 레이아웃 및 컴포넌트 스타일 최적화.

## [프로젝트 구조 및 문서화]
- **디렉토리 재구조화**: `share-king` 하위 폴더의 모든 소스를 루트 디렉토리로 이동하여 표준화.
- **Vercel 배포 자동화**: GitHub 연동 및 배포 가이드(`VERCEL_DEPLOY.md`) 작성.
- **문서 통합**: 모든 마크다운 파일(`README`, `CHANGELOG` 등)을 `docs/` 폴더로 정리.

## [버그 수정 및 개선]
- **RLS 정책 최적화**: 데이터 생성(INSERT) 권한 누락 문제를 해결하기 위한 SQL 스크립트 작성 및 반영.
- **오류 로깅 개선**: 백엔드 연동 관련 에러 상황을 명확히 파악할 수 있도록 팝업 알림 및 콘솔 로그 강화.
- **환경 변수 관리**: Supabase Credentials 설정을 위한 `.env.local` 템플릿 정리.
